# runningRhythm
支持iTunes导入音乐和图片，导入的音乐文件会在左侧滑动显示，导入的图片会用作随机背景图片。播放音乐同时可以选择时间倒计时，时间到自动停止播放音乐，可以显示本次时间和累计总时间。
